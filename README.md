# simon-game
Simon Game built using Vanilla JavaScript to practice event-driven programming, state management, and timing control. The game generates a growing color sequence that the user must repeat correctly. Focused on logic, user interaction, and core frontend fundamentals.

## Known Issues & Future Improvements

- Sound playback may feel slightly delayed on some devices.
- Planned improvements include preloading audio files and optimizing timing.
- UI and animation polish will be improved in future updates.
